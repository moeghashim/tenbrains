# Chrome Web Store Listing Draft

## Title
Tenbrains for X

## Category
Productivity

## Short Description
Analyze public X posts and save tagged insights to Tenbrains.

## Detailed Description
Tenbrains for X adds Tenbrains actions directly inside x.com. Analyze a public post inline, review the topic, summary, intent, and concept chips, then save tagged insights to your Tenbrains bookmarks without leaving the timeline.

Tenbrains for X requires a Tenbrains account connected with X. The extension is not affiliated with X Corp.

## Privacy Field Guidance
- Single purpose: Analyze public X posts and save tagged insights to Tenbrains.
- Website content handled: tweet URL, tweet text, and tweet metadata for the post you choose.
- User activity handled: bookmark tags and in-extension action state needed to resume sign-in.
- Personal identifiers handled: Tenbrains account identifiers needed to confirm sign-in.
- Remote code: No.
